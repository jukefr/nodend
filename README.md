## Usage
```bash
$ hugo          # build in docs/
$ hugo server   # local dev server
```

## Requirements
- Hugo

## Optional
- html-minifer npm package
- uglify-js npm package
- clean-css npm package

You can run `./bin/build.bash` to do a full minify of the `docs/` directory.
